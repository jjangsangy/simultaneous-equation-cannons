# Project Brief: Simultaneous Equation Cannons Calculator

## Core Requirements
A React-based web application designed to help Yu-Gi-Oh! players calculate the specific requirements for resolving the effect of the card "Simultaneous Equation Cannons".

## Goals
- Provide an intuitive interface for inputting board state (Total Cards, Opponent Rank/Level).
- Allow users to manage their Extra Deck availability (Fusion Levels and Xyz Ranks).
- Calculate valid combinations of Fusion and Xyz monsters to banish.
- Display all possible combinations for a given board state.
- Persist user preferences and inputs using LocalStorage.
- Provide a clean, modern UI using Tailwind CSS.
